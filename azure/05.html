What is a Resource Group?

An Azure Resource Group is a logical container (not a physical thing) that holds related Azure resources for a single application or solution.

Think of it as a folder in Azure that:

  • Groups resources together for easier lifecycle management
  • Lets you deploy, update, monitor, and delete a whole application in one operation
  • Applies Role-Based Access Control (RBAC) and Policies at the group level
  • Controls billing â€“ you can see costs for everything inside one resource group
What CAN be in a Resource Group What CANNOT be in a Resource Group

Key facts interviewers love:

  • A resource can belong to exactly one resource group (no sharing).
  • Deleting a resource group deletes everything inside it (no undo!).
  • Resource Groups are free â€“ no cost.
  • You can move most resources between resource groups (or even subscriptions) later.
